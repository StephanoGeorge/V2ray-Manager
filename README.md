# V2Ray 管理脚本

因图形界面客户端难以为所有连接自定义配置, 所以编写此脚本

仅支持 Linux

# 使用方法

## 首次运行

将 V2Ray 配置文件重命名为 `V2Ray-Config.json`, 并置于 `~/.config/V2Ray-Manager/` 中.

可选地, 向 PATH 中添加此脚本的符号链接:

```sh
ln -s $PWD/V2Ray-Manager.py ~/.local/bin/v2man
```

## 运行

运行此脚本
